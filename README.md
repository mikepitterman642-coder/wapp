# TredStone AI — локальный MVP сайта / мини-приложение

Отдельный локальный React + Vite прототип для AI-сервиса **TredStone AI**. Это демо-приложение для визуального тестирования: без реального сервером, без платежей, без авторизации и без внешних API.

## Запуск

```bash
cd webapp
npm install
npm run dev
```

После запуска Vite откроет локальный адрес (обычно `http://localhost:5173`).

## Где делались изменения

Работа велась **только в папке `webapp/`**:

```text
webapp/src/components  UI-компоненты, shell, brand logo, loader, toast, actions
webapp/src/pages       страницы landing/dashboard/photo/video/audio/chat/profile/articles
webapp/src/data        демо-данные моделей, ролей, фотосессий, истории и тарифов
webapp/src/lib         утилиты
webapp/index.html      title/meta/favicon brand-зона
webapp/README.md       команды запуска и описание
```

Код бота, webhook, сервером, `.env`, платежи и интеграции не изменялись.

## Страницы

- `/` — landing: hero, popular, social proof, витрина, тарифы, TredStone Core.
- `/app` — панель с балансом, лимитами и быстрыми функциями.
- `/photo` — демо-интерфейс генерации фото и галерея готовых фотосессий.
- `/video` — демо-интерфейс генерации видео и поля управления движением.
- `/audio` — демо-аудио: текст в голос и музыка.
- `/chat` — демо-чат с моделями, ролями, voice/image кнопками и индикатор ответа.
- `/tools` — галерея AI-инструментов в демо-режиме.
- `/for-her` — раздел “Для неё” с фотосессиями, питанием, внешностью и beauty/lifestyle идеями.
- `/for-him` — раздел “Для него” с фотосессиями, питанием, телом и стилем/бизнес-образом.
- `/profile` — профиль, баланс, подписка, лимиты, вкладки и история.
- `/articles` — карточки полезных материалов.

## Логотип

Если файл `webapp/src/assets/tredstone-logo.png` существует, он используется в шапке и favicon. Если файла нет, UI показывает текстовый fallback-логотип.

## Важно

- Это только красивый локальный прототип.
- Реальных API ключей нет.
- Реальных платежей нет.
- Интеграции с сервером нет.
